# Demonstration Video Structure & Content

**Project:** Automated Employee Onboarding & Offboarding System

---

## Overview

This document specifies the organization and demonstration points for a **video walkthrough of the system**. The presentation will illustrate complete system functionality spanning request submission, authorization workflows, task distribution, Service Level Agreement tracking, and request completion. The video showcases the end-to-end automation capabilities implemented in ServiceNow.

---

## Video Demonstration Sequence

## Section 1: Portal Interface Demonstration

**Access Path:** Service Portal → **Service Catalog → HR Services → Onboard / Offboard Employee**

**Presentation Highlights:**
- Introduce the Service Portal interface
- Navigate through **HR Services category** structure
- Open the **Onboard / Offboard Employee** catalog entry
- Explain the catalog form's functional purpose

---

## Section 2: Request Form Submission

**Procedure:**
1. Choose **Request Classification** (Onboarding or Offboarding)
2. Complete all required input fields:
   - Employee Identification
   - Department Assignment
   - Manager/Supervisor
   - Commencement Date / Departure Date
   - Equipment & Asset Information
   - System Access Specifications
3. Finalize and submit the form

**Presentation Highlights:**
- Demonstrate conditional form field rendering using **UI Policies**
- Illustrate how fields adapt based on request type selection

---

## Section 3: Authorization & Manager Workflow

**Presentation Highlights:**
- Demonstrate request transmission to **manager-level personnel**
- Display authorization request interface for manager review
- Execute authorization action to activate subsequent workflow stages

---

## Section 4: Automated Department Task Distribution

Following authorization, the system automatically creates tasks for:
- HR Department
- IT Department
- Facilities Department
- Security Department

**Presentation Highlights:**
- Display automatically generated task assignments
- Show task allocation and activity progression

---

## 5. SLA Monitoring

### Demonstration Points

* Show SLA timers attached to departmental tasks.
* Explain how SLAs help track task completion deadlines.
* Demonstrate SLA status updates.

---

## 6. Notifications

### Demonstration Points

Show automated **email notifications** sent to:

* Managers for approval requests
* Departments for task assignments
* Employees for status updates
* Request closure confirmation

---

## 7. Request Lifecycle Completion

### Demonstration Points

* Show completion of all departmental tasks.
* Demonstrate automatic **status updates in the Employee Lifecycle table**.
* Show final request closure.

---

# End-to-End Lifecycle Summary

The demo video should clearly show the complete lifecycle:

1. Request Submission
2. Manager Approval
3. Task Creation
4. SLA Monitoring
5. Notifications
6. Task Completion
7. Request Closure

---

## Expected Outcome of Demo

The demonstration should clearly show:

* Automation of employee lifecycle processes.
* Coordination between HR, IT, Facilities, and Security teams.
* Workflow transparency and lifecycle tracking.
* Improved efficiency through ServiceNow automation.
